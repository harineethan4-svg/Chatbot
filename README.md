This project is an end-to-end Medical Information Retrieval System built using a Retrieval-Augmented Generation (RAG) architecture. It enables users to query complex medical datasets and receive context-aware, fact-based responses by grounding the Large Language Model (LLM) in specific, verified medical literature.




Tech Stack

Language: Python (Pandas, NumPy) 

Orchestration: LangChain

Vector Database: ChromaDB


Generative AI: Retrieval-Augmented Generation (RAG) 


NLP: Advanced Text Processing and Embeddings
